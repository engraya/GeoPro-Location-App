About

IP-API has been running since 2012 and we are now providing one of the most popular and reliable IP Geolocation API. The documentation is used by developers who want to write applications that can query IP-API. Data is served in multiple formats via a simple URL-based interface over HTTP, which enables you to use our data directly from a user's browser or from your server.
